About This Project

This Fake News Detection system is an advanced web application that leverages Machine Learning and Natural Language Processing to combat misinformation. Built with cutting-edge technology, it provides instant, accurate analysis of news content to help users identify potentially false information.

The system uses a combination of supervised learning algorithms, including Passive Aggressive Classifier and TF-IDF vectorization, trained on thousands of verified news articles. Our mission is to make fact-checking accessible, fast, and reliable for everyone.
